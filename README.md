### A simple task tracker

## How to use
1. download the task_tracker_cli.jar
2. open a terminal and go to the directory of the task_tracker_cli.jar 
3. run `java -jar task_tracker_cli.jar`

## Commands:
    add
        add (description)
        add Buy coffee
    list
        list all
        list done
        list todo
        list in progress
    set
        set (id) (status)
        set 1 done
    delete
        delete (id)
        delete 1
        delete all

## language: Java
## database: JSON
