### A simple task tracker

commands:
    add
        add (description)
        add Buy coffee
    list
        list all
        list done
        list todo
        list in progress
    set
        set (id) (status)
        set 1 done
    delete
        delete (id)
        delete 1
        delete all

## language: Java
## database: JSON
